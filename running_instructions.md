# Инструкция по запуску проекта

Проект состоит из **Backend** (Golang + SQLite) и **Frontend** (React + Vite). Запускать их нужно в разных окнах терминала (PowerShell или встроенный терминал VS Code).

---

## 🚀 Шаг 1: Запуск Backend (Сервер)

Наш сервер написан на Go и использует базу данных SQLite, для компиляции которой необходим CGO и компилятор Zig.

1. Открой терминал PowerShell.
2. Перейди в папку проекта, а затем в папку сервера:
   ```powershell
   cd C:\asoiu\server
   ```
3. Запусти сервер со специальными переменными окружения:
   ```powershell
   $env:CGO_ENABLED="1"; $env:CC="zig cc"; $env:CXX="zig c++"; go run main.go
   ```
4. Если всё сделано верно, ты увидишь сообщение в консоли:
   `API Server running on http://localhost:3000 (Snake Case DB Mode)`

---

## 🚀 Шаг 2: Запуск Frontend (React UI)

Пользовательский интерфейс использует Vite, для этого нужен Node.js (v20.19+, v22.12+).

1. Открой **новое** окно терминала.
2. Перейди в корень проекта:
   ```powershell
   cd C:\asoiu
   ```
3. Выполни команду запуска:
   ```powershell
   npm run dev
   ```
4. Если вылезет сообщение о занятом порту, Vite сам предложит другой. Обычно сайт доступен по ссылке:
   `http://localhost:5173`

*(Примечание: если команда `npm` или `node` вдруг не найдена и выдает ошибку, это означает, что в текущем терминале не подгружены системные пути Node.js. Открой проект через встроенный терминал среды, например, **VS Code**, или используй **Node.js Command Prompt**).*
